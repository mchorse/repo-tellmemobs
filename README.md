# Tell Me Mobs

A simple client-side mod that lists the mobs that have spawned on the map in the truck screen. Additionally, there is a config option that enables displaying mob list in the HUD (by default disabled).

# Скажи мне мобов

Простой мод, на клиентской стороне, который перечисляет всех мобов которые заспаунились на уровне на экране грузовика. Кроме того, есть настройка, которая позволяет отображать список мобов в HUD (по умолчанию отключена).
